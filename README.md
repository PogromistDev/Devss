# Devss
![icon](web/icon.png)

Take screenshots of your computer screen remotely with a web ui accessable from any device
#### requirements
    windows operating system
    node.js
    dot net framework
#### installation
```console
npm install -g
```
#### usage
```console
devss
```
    open web browser from any device and enter <host:port> ip address to access web ui
```console
devss --help
```
    show help message
